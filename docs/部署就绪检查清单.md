## 🚀 服务器到手后（按顺序执行）

### Step 1: 连接服务器
```bash
ssh root@<服务器IP>
# 或使用腾讯云控制台WebSSH
```

### Step 2: 安装环境
```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs

# 安装 Nginx
apt install -y nginx

# 安装 PM2 (进程管理)
npm install -g pm2

# 验证
node -v  # 应显示 v20.x.x
nginx -v # 应显示 nginx version
```

### Step 3: 上传代码
```bash
# 在本地执行
scp -r /Users/x/.openclaw/workspace/ai-temple root@<服务器IP>:/var/www/
```

### Step 4: 配置后端
```bash
# 在服务器上
cd /var/www/ai-temple
npm install --production
pm2 start server.js --name ai-temple
pm2 save
pm2 startup  # 按提示执行
```

### Step 5: 配置 Nginx
```bash
# 创建配置
cat > /etc/nginx/sites-available/ai-temple << 'EOF'
server {
    listen 80;
    server_name temple.ai www.temple.ai;
    
    location / {
        root /var/www/ai-temple;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
EOF

# 启用配置
ln -s /etc/nginx/sites-available/ai-temple /etc/nginx/sites-enabled/
nginx -t  # 测试配置
systemctl restart nginx
```

### Step 6: 配置域名
```
在腾讯云DNS控制台：
- 添加 A 记录：temple.ai → 服务器IP
- 添加 A 记录：www.temple.ai → 服务器IP
```

### Step 7: HTTPS (Let's Encrypt)
```bash
# 安装 certbot
apt install -y certbot python3-certbot-nginx

# 申请证书
certbot --nginx -d temple.ai -d www.temple.ai

# 自动续期
certbot renew --dry-run
```

### Step 8: 验证
```bash
curl http://localhost:3000/api/ais  # 后端API
curl http://localhost               # Nginx前端
curl https://temple.ai              # 域名访问
```

## 🆘 常见问题

### 端口被占用
```bash
lsof -i :3000
kill -9 <PID>
```

### Nginx 502 Bad Gateway
- 检查后端是否运行：`pm2 status`
- 检查端口是否正确：`netstat -tlnp | grep 3000`

### 域名无法访问
- 检查DNS解析：`nslookup temple.ai`
- 检查Nginx配置：`nginx -t`
- 检查防火墙：`ufw status`
